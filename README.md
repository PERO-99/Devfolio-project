# Devfolio-project
# 🚀 VERITAS X — AI Trust & Threat Detection Platform

> A real-time AI-powered digital security layer that detects scams before users fall for them.

---

## 🔴 Problem

Online scams are rapidly increasing, especially in regions like India.
Users face threats like:

* OTP fraud (“Share your OTP” scams)
* UPI/payment fraud
* Fake KYC verification links
* Phishing websites
* Lottery/reward scams

⚠️ **Problem:**
Users cannot identify scams in real-time while browsing.

---

## 💡 Solution

**VERITAS X** acts as a real-time protection system that:

* Scans websites automatically
* Detects scam patterns instantly
* Analyzes text, images, and videos
* Warns users before damage happens

---

## 🧠 Core Features

### 🔍 Real-Time Detection

* Automatic website scanning
* Instant risk analysis

### 🧠 AI-Powered Analysis

* Text, image, and video scanning
* Pattern recognition (OTP, UPI, phishing)

### 🚨 Smart Alerts

* SAFE → No alert
* SUSPICIOUS → Warning
* DANGER → Full-screen alert

### 🌐 Chrome Extension

* Auto-scan on page load
* Scan selected text
* Capture images & video frames

### 📊 Live Dashboard

* Real-time threat feed
* Analytics & statistics
* Scan studio (manual testing)

## 🧩 Architecture

1. User opens a website
2. Extension extracts data
3. Sends request to backend
4. Backend analyzes content
5. Returns risk score + explanation
6. Alert shown to user

---

## 🎯 Risk Scoring System

* **0–30 → SAFE**
* **31–69 → SUSPICIOUS**
* **70–100 → DANGER**

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/veritas-x.git

# Navigate to project
cd veritas-x

# Install backend dependencies
pip install -r requirements.txt

# Run backend server
uvicorn main:app --reload

# Frontend setup
cd frontend
npm install
npm run dev
```
